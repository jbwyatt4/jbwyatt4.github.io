Godot Projects

A listing of Godot projects I have done.

Please note: some of the projects only work with a keyboard. Use arrow keys and x for action.

2D WebGL + WebAssembly Game

A fun touch Match 3 demo project that you can run in Firefox or Google Chrome. Based on Mister Tafts Creates Match 3 Tutorial.

Mini Platform Game

A small 2D platformer game with levels and warping. Based on HeartBeast's Platformer Tutorial.

Link
Zelda-Like Top Down Adventure Game

A Zelda-like with combat, enemies, keys, locked doors, and heart pickups. Based on fornclake's Zelda-like Tutorial.

Link
2.5D Isometric Road Map Generator

Generates a new random road map with every reload. Based on KidsCanCode's Procedural Content Generation Tutorials.

Link
3 Hour TryJam Demo

How long can you avoid the evil knights? My first original solo developed game for the 3 Hour TryJam game jam. Developed on Manjaro Linux with Asperite for pixel art and Godot for the game engine.

Link
1-Bit Mobile Touch Dungeon Crawler

Attack evil slimes and giant rats. A very old school 1-Bit RPG example that demos Godot's theming and UI capabilities with a touch interface. Based on HeartBeast's Godot Mobile RPG tutorial.
